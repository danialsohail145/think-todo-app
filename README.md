## Run Locally

Clone the project

```bash
  git clone https://github.com/danialsohail145/think-todo-app
```

Go to the project directory

```bash
  cd think-todo-app
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  ng serve
```
